# Notes on contents
This repo contains **only our agent code**. It excludes the competition engine
binaries (`cg/`, `libcg.so`, `cg.dll`), the card reference PDFs/CSVs, and tables
derived from them — provided by The Pokémon Company / Matsuo Institute via the
Kaggle competition page; do not redistribute. Download them from the competition
and generate tables with `inspect_cards.py` (see README).
